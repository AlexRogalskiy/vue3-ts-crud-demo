# Vue 3 Typescript Vite CRUD DEMO

## Introduction

Vue 3 + Typescript + Vite crud template project

## Technology stack

- Vue 3
- Vue Router 4
- Typescript
- Vite

## Usage

install
```shell
yarn
```
develop
```shell
yarn dev
```
build
```shell
yarn build
```
